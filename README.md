# MoodMeal — Data-Driven Business Intelligence Dashboard

A Streamlit-powered analytics dashboard for **MoodMeal**, an Indian D2C food startup offering personalised meal bowls, mood-based functional beverages, comfort food combos, breakfast boxes, and office lunch packs.

## Features
- **EDA & Overview** — Interactive exploration of 7,500 survey respondents
- **Customer Segmentation** — K-Means clustering with Elbow & Silhouette analysis
- **Purchase Intent Classification** — Random Forest, XGBoost, Logistic Regression with ROC curves
- **Association Rule Mining** — Apriori algorithm finding product-mood-cuisine affinities
- **Spending Prediction** — Regression models predicting AOV and monthly spend
- **Prescriptive Strategy** — Auto-generated business recommendations
- **New Customer Predictor** — Upload CSV to predict new customers' MoodMeal inclination

## Tech Stack
Python, Streamlit, Scikit-learn, XGBoost, MLxtend, Plotly, Pandas

## Deployment
Hosted on [Streamlit Community Cloud](https://streamlit.io/cloud)
